## 🌸 little wonders

- 🔐 User signup & login *(JWT auth)*
- ✍️ Add, delete & update tasks
- 🎀 Mark tasks as completed
- 🌷 Priority levels — high, medium, low
- 📊 Weekly productivity chart
- 🔥 Streak tracking
- 🗒️ Notes & due dates
